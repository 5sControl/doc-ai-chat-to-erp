# Поддержка Markdown в карточках знаний

## Обзор

Карточки знаний теперь поддерживают отображение Markdown-форматирования, что позволяет отображать текст с форматированием: жирный текст, курсив, списки, заголовки и т.д.

## Изменения

### Обновленные файлы

1. **knowledge_card_tile.dart** - Превью карточки в списке
2. **card_detail_modal.dart** - Детальное отображение карточки в модальном окне
3. **saved_card_tile.dart** - Отображение сохраненных карточек
4. **demo_knowledge_cards.dart** - Обновлены демонстрационные карточки с примерами Markdown

### Технические детали

- Использован пакет `flutter_markdown` (уже был в зависимостях)
- Виджет `Text` заменен на `MarkdownBody`
- Настроены стили для соответствия дизайну приложения
- Добавлена поддержка:
  - **Жирного текста** через `**текст**`
  - *Курсива* через `*текст*`
  - Нумерованных списков через `1. пункт`
  - Маркированных списков через `- пункт`
  - `Моноширинного текста` через обратные кавычки
  - Заголовков через `# Заголовок`

## Примеры использования

### Жирный текст
```markdown
**Важная информация**
```

### Курсив
```markdown
*Пример формулы*
```

### Нумерованный список
```markdown
1. **Obvious** (Cue) - Make the cue visible
2. **Attractive** (Craving) - Make it appealing
3. **Easy** (Response) - Reduce friction
4. **Satisfying** (Reward) - Make it immediately rewarding
```

### Маркированный список
```markdown
- **Goals** are about results you want
- **Systems** are about the processes that lead to those results
```

### Комбинированное форматирование
```markdown
**The Two-Minute Rule**: Downscale any new habit to take two minutes or less to do.

*"Go for a run"* becomes *"put on running shoes."*

Master the art of showing up.
```

## Настройка стилей

Стили Markdown настроены в каждом виджете отображения карточек через `MarkdownStyleSheet`:

- `p` - обычный параграф
- `strong` - жирный текст
- `em` - курсив
- `code` - моноширинный текст с фоном
- `h1`, `h2`, `h3` - заголовки разных уровней
- `listBullet` - маркеры списков

## Демонстрационные карточки

Обновлены демонстрационные карточки в `demo_knowledge_cards.dart`, чтобы показать примеры использования Markdown-форматирования. Пользователи могут увидеть эти примеры в демо-режиме приложения.

## Обратная совместимость

Карточки без Markdown-форматирования (обычный текст) продолжат отображаться корректно, так как `MarkdownBody` обрабатывает обычный текст как есть.
