# Summify

Personal AI Summarizer — summarize and translate content in one tap.

## Supported languages (interface localization)

The app UI is localized in: **English**, **Spanish**, **French**, **German**, **Italian**, **Polish**, **Portuguese**, **Russian**, **Turkish**, **Vietnamese**, **Chinese (Simplified)**.

Language selection at onboarding and in Settings uses the same list; Turkish (Türkçe) is available with its flag icon.
