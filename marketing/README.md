# Marketing

Материалы для маркетинга и сторисейлов Summify.

## Структура

### `lang/` — тексты и картинки (EN/RU)

| Файл | Назначение |
|------|------------|
| **play-store-full-description.txt** | Полное описание для Google Play (Full description, до 4000 символов). Фичи: суммаризация, карточки, квиз, перевод, TTS, диаграммы. |
| **learn-and-study-with-summify.md** | Черновик статьи для блога: зачем учить язык и учиться с Summify (EN). |
| **instant-translation-article.md** | Статья про мгновенный перевод (EN). |
| **instant-translation-article.ru.md** | То же, русская версия. |
| **translate-problem-flow.png** | Картинка: проблема (flow перевода). |
| **translate-problem-flow.ru.png** | То же, RU. |
| **translate-solution-instant.png** | Картинка: решение — мгновенный перевод. |
| **translate-solution-instant.ru.png** | То же, RU. |

## Store listings

- **Google Play:** Short description (80 символов) — см. обсуждения в чате; Full description — скопировать из `lang/play-store-full-description.txt`.
- **App Store:** Name (30), Subtitle (30) — см. обсуждения в чате (Learn Languages on Doc Summify / Cards Quiz TTS GPT Translate).

## Обновления

При добавлении новых статей, скриншотов или вариантов описаний класть в `lang/` и при необходимости обновить этот README.
