# Promo Video Prompt: Карточки знаний

## Цель видео
Коротко и визуально объяснить новую функцию **«Карточки знаний»**: как пользователь открывает карточки из summary, фильтрует по типам, сохраняет важные карточки, открывает детали, копирует и делится.

## Формат
- Длительность: 20-35 секунд
- Вертикаль: 9:16 (App Store preview/social)
- Стиль: clean, premium, AI-product
- Ритм: быстрые понятные переходы, акцент на UI

## Универсальный шаблон сцены (повторять для каждого шага)
1. **Текст-заголовок по центру** (по высоте и ширине, 1-2 строки)
2. **Появление соответствующего экрана приложения**
3. **Подсветка области экрана**, которая подтверждает текст
4. **Короткий переход** к следующей сцене (zoom/slide/fade)
